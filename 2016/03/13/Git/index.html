记录一些日常的git操作

发表于 2016-03-13   |   分类于 Git 日常   |  

GitUse

1. 提交

git add . //把所有文件 添加到本地repository
git commit -m //提交代码到本地代码库

2. 同步

git pull origin master //同步到服务器前先需要将服务器代码同步到本地

3. 远程库

git remote add origin git@github.com:akvsdk/OkHttp.git //关联一个远程库

4. 推送

git push origin master //推送代码到远程库

5. 回滚

git reflog //可以查询出每个历史版本的commit，每个commit都有对应的ID。
git reset –hard xxxx // 确认好要退回的ID后输入

6.移除版本控制

git remote rm origin // 移除远程仓库
git rm -r –cached . // 将所有代码移除版本控制
